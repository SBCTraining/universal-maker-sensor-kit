Instalación del IDE de Arduino (Importante)
================================================

El IDE de Arduino, conocido como Entorno de Desarrollo Integrado de Arduino, proporciona todo el soporte de software necesario para completar un proyecto de Arduino. Es un software de programación específicamente diseñado para Arduino, proporcionado por el equipo de Arduino, que nos permite escribir programas y subirlos a la placa Arduino.

El IDE de Arduino 2.0 es un proyecto de código abierto. Representa un gran paso respecto a su robusto predecesor, el IDE de Arduino 1.x, y viene con una interfaz renovada, mejor gestión de placas y bibliotecas, depurador, función de autocompletado y mucho más.

En este tutorial, mostraremos cómo descargar e instalar el IDE de Arduino 2.0 en tu computadora con Windows, Mac o Linux.

Requisitos
-------------------

* Windows - Win 10 y más reciente, 64 bits
* Linux - 64 bits
* Mac OS X - Versión 10.14: "Mojave" o más reciente, 64 bits

Descarga del IDE de Arduino 2.0
-------------------------------

#. Visita |link_download_arduino|.

#. Descarga el IDE para la versión de tu sistema operativo.

   .. image:: img/install_ide_01.png

Instalación
------------------------------

Windows
^^^^^^^^^^^^^

#. Haz doble clic en el archivo ``arduino-ide_xxxx.exe`` para ejecutar el archivo descargado.

#. Lee el Acuerdo de Licencia y acéptalo.

   .. image:: img/install_ide_02.png

#. Elige las opciones de instalación.

   .. image:: img/install_ide_03.png

#. Elige la ubicación de instalación. Se recomienda instalar el software en una unidad distinta a la unidad del sistema.

   .. image:: img/install_ide_04.png

#. Luego finaliza. 

   .. image:: img/install_ide_05.png

macOS
^^^^^^^^^^^^^^^^

Haz doble clic en el archivo ``arduino_ide_xxxx.dmg`` descargado y sigue las instrucciones para copiar la **aplicación Arduino IDE** en la carpeta **Aplicaciones**. Verás que la IDE de Arduino se ha instalado con éxito después de unos segundos.

.. image:: img/install_ide_macos.png
    :width: 100%

Linux
^^^^^^^^^^^^

Para el tutorial sobre cómo instalar el IDE de Arduino 2.0 en un sistema Linux, consulta: https://docs.arduino.cc/software/ide-v2/tutorials/getting-started/ide-v2-downloading-and-installing#linux


Abrir el IDE
--------------

#. Cuando abras el IDE de Arduino 2.0 por primera vez, se instalarán automáticamente las Placas Arduino AVR, las bibliotecas incorporadas y otros archivos necesarios.

   .. image:: img/install_ide_06.png

#. Además, tu firewall o centro de seguridad puede aparecer algunas veces preguntándote si deseas instalar algunos controladores de dispositivo. Por favor, instala todos ellos.

   .. image:: img/install_ide_07.png

#. ¡Ahora tu IDE de Arduino está listo!

   .. note::
       En el caso de que algunas instalaciones no funcionen debido a problemas de red u otras razones, puedes volver a abrir el IDE de Arduino y este completará el resto de la instalación. La ventana de salida no se abrirá automáticamente después de que todas las instalaciones estén completas a menos que hagas clic en Verificar o Subir.
