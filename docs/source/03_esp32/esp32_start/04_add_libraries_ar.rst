Installa librerie (Importante)
======================================

Una libreria è una raccolta di codice pre-scritto o funzioni che estendono le capacità dell'IDE Arduino. Le librerie forniscono codice pronto all'uso per varie funzionalità, permettendoti di risparmiare tempo e sforzo nella codifica di caratteristiche complesse.

Installa dal Gestore delle Librerie
----------------------------------------

Numerose librerie sono disponibili direttamente tramite il Gestore delle Librerie Arduino. Puoi accedere al Gestore delle Librerie seguendo questi passi:

#. Nel **Gestore delle Librerie**, puoi cercare la libreria desiderata per nome o sfogliare diverse categorie.

   .. note::

      Nei progetti in cui è richiesta l'installazione di una libreria, ci saranno indicazioni sulle librerie da installare. Segui le istruzioni fornite, come "Per installare la libreria, utilizza il Gestore delle Librerie Arduino e cerca 'libreria sensore DHT' e installala." Cerca e installa le librerie raccomandate come indicato.

   .. image:: img/install_lib3.png

#. Una volta trovata la libreria che desideri installare, clicca su di essa e poi sul pulsante **Installa**.

   .. image:: img/install_lib2.png

#. L'IDE Arduino scaricherà e installerà automaticamente la libreria per te.
