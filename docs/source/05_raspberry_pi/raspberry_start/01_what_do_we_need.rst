Di Cosa Abbiamo Bisogno?
===========================

Componenti Necessari
-------------------------

**Raspberry Pi**

Il Raspberry Pi è un computer a basso costo, delle dimensioni di una carta di credito, che si collega a un monitor o a una TV e si utilizza con tastiera e mouse standard. È un dispositivo versatile che consente a persone di tutte le età di esplorare l'informatica e imparare linguaggi di programmazione come Scratch e Python.

**Scheda Micro SD**

Il tuo Raspberry Pi ha bisogno di una scheda Micro SD per memorizzare tutti i file e il sistema operativo Raspberry Pi OS. È necessaria una scheda con almeno 8 GB di capacità.

Componenti Opzionali
-------------------------

**Schermo**

Per accedere all’ambiente desktop del Raspberry Pi, puoi collegarlo a una TV o a un monitor. Se lo schermo ha altoparlanti integrati, l’audio verrà riprodotto tramite questi.

**Mouse e Tastiera**

Se utilizzi uno schermo, avrai bisogno anche di una tastiera USB e di un mouse USB.

**HDMI**

Il Raspberry Pi è dotato di porte HDMI (o Micro HDMI), compatibili con la maggior parte delle TV e dei monitor moderni. Se il tuo schermo ha solo una porta DVI o VGA, avrai bisogno di un adattatore corrispondente.

**Custodia**

Puoi inserire il tuo Raspberry Pi in una custodia per proteggerlo. Sul nostro sito ufficiale offriamo prodotti correlati in vendita; puoi visualizzare o acquistare custodie per Raspberry Pi qui: |link_buy_pi_case|.

**Casse o Cuffie**

La maggior parte dei modelli Raspberry Pi dispone di una porta audio da 3,5 mm, utile se il tuo schermo non ha altoparlanti integrati o non è in uso. Tuttavia, è importante notare che il nuovo Raspberry Pi 5 non dispone della porta audio da 3,5 mm.
