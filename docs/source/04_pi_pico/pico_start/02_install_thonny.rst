Installazione dell'IDE Thonny
=======================================

Prima di poter iniziare a programmare il Pico con MicroPython, hai bisogno di un ambiente di sviluppo integrato (IDE), qui raccomandiamo Thonny. Thonny viene fornito con Python 3.10 integrato, è necessario solo un semplice installer per essere pronti ad imparare a programmare.


.. note::

    Se hai già installato Thonny, assicurati che sia la versione 3.3.3 o successiva, poiché l'interprete Raspberry Pi Pico è compatibile solo con queste versioni; in caso contrario, aggiorna o installa la versione appropriata.

#. Puoi scaricarlo visitando il sito |link_thonny|. Una volta aperta la pagina, vedrai un riquadro grigio chiaro nell'angolo in alto a destra, clicca sul link che corrisponde al tuo sistema operativo.

   .. image:: img/download_thonny.png

#. Dopo il download, esegui l'installer per iniziare il processo di installazione. Poi clicca su "Avanti" seguito da "Installa" per completare l'installazione di Thonny.

    .. image:: img/install_thonny6.png
