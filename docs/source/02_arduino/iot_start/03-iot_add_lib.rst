.. _iot_add_library:

1.3 Ajout des bibliothèques requises
=======================================

Vous devez ajouter les bibliothèques correctes pour que l'IDE Arduino puisse utiliser Blynk.

#. Cliquez sur |link_blynk_lib|, faites défiler vers le bas jusqu'à **"Assets"** et téléchargez le premier fichier ``.zip``. 

   .. note::
    Veuillez noter que le numéro de version montré dans l'image ci-dessous pourrait être obsolète. Nous recommandons vivement de télécharger et d'installer la version la plus récente disponible.

   .. image:: img/new/add_lib_shadow.png

#. Décompressez ce fichier puis entrez dans le dossier ``libraries`` pour voir les dossiers suivants.

   .. image:: img/new/add_lib_0_shadow.png
    
#. Copiez-les tous et ajoutez-les au dossier ``libraries`` de votre carnet de croquis.

   **Étape 1 :** Vous pouvez trouver ou changer l'emplacement de votre dossier de bibliothèques dans ``File > Preferences > Sketchbook location``.

   .. image:: img/new/add_lib_1_shadow.png

   **Étape 2 :** Rendez-vous à l'emplacement de votre carnet de croquis (trouvé à partir de l'IDE Arduino). Trouvez le dossier ``libraries``, cliquez pour l'ouvrir.

   .. image:: img/new/add_lib_2_shadow.png

   **Étape 3 :** Collez tous les dossiers décompressés de ``Blynk_Release_vx.x.x\libraries`` dans le dossier de bibliothèques.

   .. image:: img/new/add_lib_3_shadow.png
