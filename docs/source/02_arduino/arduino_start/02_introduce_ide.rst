Introduzione all'Arduino IDE
=================================

.. image:: img/sp_ide_2.png

1. **Verifica**: Compila il tuo codice. Eventuali errori di sintassi verranno segnalati.

2. **Carica**: Carica il codice sulla tua scheda. Quando clicchi sul pulsante, i LED RX e TX sulla scheda lampeggeranno rapidamente finché il caricamento non sarà completato.

3. **Debug**: Controllo degli errori riga per riga.

4. **Seleziona Scheda**: Configurazione rapida della scheda e della porta.

5. **Serial Plotter**: Visualizza l’andamento dei valori letti nel tempo.

6. **Monitor Seriale**: Apre una finestra che riceve i dati inviati dalla scheda di controllo. Molto utile per il debug.

7. **File**: Clicca per aprire il menu a discesa che include operazioni come creare, aprire, salvare, chiudere file e configurare parametri.

8. **Modifica**: Menu per operazioni di editing come **Taglia**, **Copia**, **Incolla**, **Trova**, ecc., con relative scorciatoie.

9. **Sketch**: Contiene operazioni come **Verifica**, **Carica**, **Aggiungi file**, ecc. La funzione più importante è **Includi Libreria** – per aggiungere librerie al tuo progetto.

10. **Strumenti**: Include strumenti utili – in particolare la selezione della **Scheda** e della **Porta**, fondamentali prima di caricare il codice.

11. **Aiuto**: Se sei un principiante, qui puoi trovare assistenza su operazioni dell’IDE, informazioni introduttive, risoluzione problemi, spiegazioni del codice, ecc.

12. **Barra di Output**: Cambia la scheda dell’output.

13. **Finestra di Output**: Visualizza informazioni di stampa o di log.

14. **Scheda e Porta**: Anteprima della scheda e porta selezionate per il caricamento. Puoi modificarle in **Strumenti** -> **Scheda** / **Porta** se necessario.

15. Area di scrittura dell’IDE. Qui puoi scrivere il tuo codice.

16. **Sketchbook**: Gestione dei file sketch.

17. **Gestore Schede**: Per installare o aggiornare i driver delle schede.

18. **Gestore Librerie**: Per gestire le librerie utilizzate nei tuoi progetti.

19. **Debug**: Per il debug del codice.

20. **Cerca**: Cerca all’interno del codice dei tuoi sketch.
