Scarica e installa Arduino IDE 2.0
========================================

L’Arduino IDE, ovvero l’Ambiente di Sviluppo Integrato Arduino, fornisce tutto il supporto software necessario per realizzare un progetto Arduino. È un software di programmazione progettato specificamente per Arduino e sviluppato dal team Arduino. Consente di scrivere programmi e caricarli sulla scheda Arduino.

Arduino IDE 2.0 è un progetto open-source. Rappresenta un grande passo avanti rispetto al solido predecessore Arduino IDE 1.x e introduce una nuova interfaccia grafica, un gestore di schede e librerie migliorato, debugger integrato, completamento automatico del codice e molto altro.

In questo tutorial ti mostreremo come scaricare e installare Arduino IDE 2.0 sul tuo computer Windows, Mac o Linux.

Requisiti
-------------------

* Windows - Windows 10 o successivo, 64 bit
* Linux - 64 bit
* Mac OS X - Versione 10.14 "Mojave" o successiva, 64 bit

Scarica Arduino IDE 2.0
-------------------------------

#. Visita |link_download_arduino|.

#. Scarica la versione dell’IDE adatta al tuo sistema operativo.

   .. image:: img/sp_001.png

Installazione
------------------------------

Windows
^^^^^^^^^^^^^

#. Fai doppio clic sul file ``arduino-ide_xxxx.exe`` scaricato per avviare l’installazione.

#. Leggi il contratto di licenza e accettalo.

   .. image:: img/sp_002.png

#. Scegli le opzioni di installazione.

   .. image:: img/sp_003.png

#. Scegli la cartella di installazione. Si consiglia di installare il software su un’unità diversa da quella di sistema.

   .. image:: img/sp_004.png

#. Infine, clicca su Finish.

   .. image:: img/sp_005.png

macOS
^^^^^^^^^^^^^^^^

Fai doppio clic sul file ``arduino_ide_xxxx.dmg`` scaricato e segui le istruzioni per copiare **Arduino IDE.app** nella cartella **Applicazioni**. Dopo pochi secondi vedrai che Arduino IDE è stato installato correttamente.

.. image:: img/macos_install_ide.png
    :width: 800

Linux
^^^^^^^^^^^^

Per il tutorial su come installare Arduino IDE 2.0 su un sistema Linux, fai riferimento a |link_install_arduino_linux|


Avvio dell’IDE
-----------------

#. Quando avvii Arduino IDE 2.0 per la prima volta, verranno installate automaticamente le Arduino AVR Boards, le librerie integrate e altri file richiesti.

   .. image:: img/sp_901.png

#. Inoltre, il tuo firewall o centro sicurezza potrebbe mostrare delle notifiche chiedendo se installare alcuni driver. Conferma l’installazione di tutti i componenti.

   .. image:: img/sp_104.png

#. Ora il tuo Arduino IDE è pronto all’uso!

   .. note::
     Nel caso in cui alcune installazioni non vadano a buon fine a causa di problemi di rete o altri motivi, puoi riaprire Arduino IDE e il programma completerà il resto dell’installazione. La finestra di Output non si aprirà automaticamente al termine, a meno che tu non clicchi su Verifica o Carica.
