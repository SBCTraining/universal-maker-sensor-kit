Placa de Pruebas
===================

.. image:: img/38_breadboard.png
    :width: 600

Una placa de pruebas es una base de construcción para la creación de prototipos electrónicos. Originalmente, la palabra se refería a una tabla literal para cortar pan, una pieza de madera pulida utilizada para rebanar pan. En la década de 1970, se hizo disponible la placa de pruebas sin soldadura (también conocida como placa de conexiones, o placa de terminales) y hoy en día el término "placa de pruebas" se usa comúnmente para referirse a estas.

Se utiliza para construir y probar circuitos rápidamente antes de finalizar el diseño de 
cualquier circuito. Tiene muchos agujeros en los que se pueden insertar componentes como ICs, 
resistores y cables puente. La placa de pruebas permite insertar y retirar componentes con facilidad.

La imagen muestra la estructura interna de una placa de pruebas. Aunque estos agujeros en la placa parecen estar independientes entre sí, en realidad están conectados entre sí a través de tiras metálicas internamente.
