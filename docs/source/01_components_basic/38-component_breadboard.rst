Platine d'essai
==================

.. image:: img/38_breadboard.png
    :width: 600

Une platine d'essai est une base de construction pour le prototypage électronique. À l'origine, le terme désignait une planche à pain, un morceau de bois poli utilisé pour trancher le pain. Dans les années 1970, la platine d'essai sans soudure (aussi connue sous le nom de plugboard, une carte à bornes) est devenue courante, et de nos jours, le terme "platine d'essai" est couramment utilisé pour désigner ces dispositifs.

Elle est utilisée pour construire et tester rapidement des circuits avant de finaliser une conception de circuit.
Elle possède de nombreux trous dans lesquels peuvent être insérés des composants comme des CI et des résistances, ainsi que des fils cavaliers.
La platine d'essai permet de brancher et de retirer facilement les composants.

L'image montre la structure interne d'une platine d'essai.
Bien que ces trous sur la platine d'essai semblent être indépendants les uns des autres, ils sont en réalité connectés entre eux par des bandes métalliques à l'intérieur.
